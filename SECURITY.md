# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately by emailing the maintainer rather than opening a public issue.

Contact: [Danila Alferov](https://stabalmo.pro)

## What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response

You can expect an initial response within 48 hours. We will work with you to understand the issue and address it promptly.

## Scope

This policy applies to all repositories in the PongCentrifugo organization:
- frontend
- backend
- iac
- terraform
- overview
